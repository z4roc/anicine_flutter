# AniCineHome

Small educational project used to practice writing TV and mobile apps with flutter

Based on https://github.com/itzzzme/anime-api, you need to host it yourself. 
Only provides existing data to a frontend

## Setup

Place a `.env`-File in the directory of the apps one the same level of their respective `pubspec.yaml`
It should look like `.env.example`.

## Install on TV

Build the TV APK and install it via adb. USB-Debugging has to be enabled on your TV for that.
